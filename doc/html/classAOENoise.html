Detailed Description

The amortized orthogonal encryption class with noise.

Extends the functionalities of the class AOE by adding a random noise parameter to the encryption and token generation steps. The system returns true in predicate decryption operations if: the token is generated with a good vector of attributes; the noise parameter used during decryption match the one used in encryption.
